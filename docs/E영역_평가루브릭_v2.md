# 부록 A. 종합 판정 공식 (자동 집계용)

```
학년군_가중치 = { L: {A:0.4, B:0.6}, M: {A:0.3, B:0.3, C:0.4}, H: {A:0.2, B:0.25, C:0.3, D:0.25} }

축_평균(축) = 해당 축에 속한 준거들의 수준 값 평균  (0~4)

종합수준 = Σ (학년군_가중치[축] × 축_평균[축])

종합판정:
  종합수준 < 1.0  → 미수행/초기
  1.0 ≤ 수준 < 2.0 → 1수준(시작)
  2.0 ≤ 수준 < 3.0 → 2수준(기초)
  3.0 ≤ 수준 < 3.7 → 3수준(도달)
  3.7 ≤ 수준      → 4수준(확장)

artifact_완성 = submit.artifact가 생성됨 + template 필드 모두 채워짐
→ 완성 시 종합 수준 하한을 2로 보장
```
