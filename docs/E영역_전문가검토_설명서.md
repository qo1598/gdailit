# E영역(Engaging) 전문가 검토 설명서

## 1. 배경

본 프로젝트 **LearnAILIT**는 OECD AI Literacy Framework(AILit)를 기반으로 한
**초등학생용 AI 리터러시 학습 플랫폼**입니다.

- 전체 구조: **4영역 × 4미션 × 3학년군(L/M/H) = 48개 카드**
- 이번 검토 대상: **E영역(Engaging) 12개 카드**

### E영역 4미션
| 코드 | 미션명 | 핵심 초점 |
|---|---|---|
| E-1 | 생활 속 AI 찾기 | 일상 속 AI 알아차리기 |
| E-2 | AI 답, 믿어도 될까? | 출력의 정확성 점검 |
| E-3 | 왜 나한테 이걸 보여줄까? | 추천·개인화 이해 |
| E-4 | 모두에게 똑같이 잘할까? | 편향·공정성 인식 |

### 학년군
- **L**: 1–2학년 (찾기·고르기·구별)
- **M**: 3–4학년 (설명·비교·간단 수정)
- **H**: 5–6학년 (분석·판단·설계)

---

## 2. 검토 중점 6가지

아래 6개 관점에서 **각 카드(12개)를 1~5점 척도**로 평정하고,
**개선이 필요한 부분**을 자유 서술로 남겨 주세요.

### 관점 1. AILit 정합성
- 카드에 명시된 **K(Knowledge) / S(Skills) / A(Attitudes)** 매핑이
  실제 미션 활동과 일관되게 연결되어 있는가?
- 명시된 K/S/A를 **실제 수행 과정에서 학습자가 경험**하게 되는가,
  아니면 라벨만 붙어 있는가?
- E영역의 핵심 개념(인식·판단·비판적 수용)이 충분히 반영되어 있는가?

### 관점 2. 학년군 적합성
- L/M/H 각 학년군의 발달 수준에 맞는 **문장 난이도·어휘·추상성**인가?
- 같은 미션이 L→M→H로 갈수록 **나선형 심화**(찾기 → 설명 → 판단·설계)
  구조가 명확히 드러나는가?
- **독해 부담**이 학년군에 비해 과하지 않은가? (특히 L의 인트로·질문)

### 관점 3. 수행 과제의 명료성
- STEP별 지시문이 **혼자서도 이해 가능**한가?
- 학생 질문·학생 활동·입력 방식이 **서로 모순 없이 정렬**되어 있는가?
- 빠진 STEP이나 과도하게 많은 STEP은 없는가? (L:3–4, M:4–5, H:5–6 권장)

### 관점 4. 초등 단독 수행 가능성
- **교사 설명·토론·발표를 전제하지 않고**도 완결되는가?
- 선택지·버튼·시각 자료 중심으로 구성되어 있는가?
- 긴 서술형 입력이 학년 수준 대비 과도하지 않은가?

### 관점 5. 평가 가능성 (JSON 수집 데이터)
- `수집 데이터(JSON)` 필드가 **학습자의 수행을 실제로 식별 가능**한가?
- 모든 STEP 응답이 JSON에 **누락 없이 반영**되어 있는가?
- 학습자 개인 판단이나 자유 응답이 **구조화된 형태**로 수집되는가?

## 3. 미션별 추가 검토 포인트

### E-1 · 생활 속 AI 찾기 (TD 탐색·구별형)
- AI/비AI 구분 예시가 **현재 초등학생이 실제로 접하는 기술**인가?
- "AI가 들어 있다"의 기준이 학년 수준에서 **혼란을 주지 않는가**?

### E-2 · AI 답, 믿어도 될까? (SJ 서술·판단형)
- 제시되는 AI 오류 사례가 **사실 검증 가능하고 재현 가능**한가?
- 학생이 "믿을지 말지"를 판단하는 **기준 세트**가 명시되어 있는가?

### E-3 · 왜 나한테 이걸 보여줄까? (SJ 서술·판단형)
- 추천·개인화 개념이 학년에 맞게 **단순화되되 왜곡되지 않는가**?
- 알고리즘이 **"내 데이터를 보고 결정"** 한다는 점이 드러나는가?

### E-4 · 모두에게 똑같이 잘할까? (SJ 서술·판단형)
- 편향 예시가 **특정 집단을 대상화**하지 않고 제시되는가?
- "공정하지 않을 수 있다"는 메시지가 **포기·냉소**가 아닌
  **개선 의지**로 연결되는가?

## 4. 검토 결과 회신 형식

각 카드별로 다음 형식으로 정리해 주시면 감사하겠습니다.

```
[E-1-L]
- 정합성: 4 / 적합성: 5 / 명료성: 3 / 단독수행: 5 / 평가성: 4 / 윤리: 5
- 핵심 의견:
  · (개선 필요) STEP 3의 장소 선택지에 "유치원" 추가 검토
  · (좋음) 비AI 대조 예시(계산기·전등) 배치가 효과적
- 기타 제안: ...
```

**종합 의견**(E영역 전체 관점): 4미션 간 균형, 학년군 나선형의 자연스러움, 다른 영역(C/M/D)과 중복·충돌 가능성 등에 대해서도 자유롭게 남겨 주세요.
