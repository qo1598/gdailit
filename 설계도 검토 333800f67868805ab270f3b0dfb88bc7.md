## **영역 1: AI 알아보기 (Engaging) - 배국환**

### **[E-1] 내 주변 숨은 AI 찾기 대작전!**

- **연결 역량**
    - **[K]**: **K1.4**: 생성, 예측, 추천 등 목적에 따라 다르게 작동하는 AI 시스템의 특성 이해
    - **[S]**: **Self and Social Awareness**: AI가 개인의 선택과 관계 및 공동체에 어떻게 영향을 미치는지 인식
    - **[A]**: **Curious**: AI 능력을 탐구하고 미래 발전을 탐색하고자 하는 호기심
- **측정 목표**: 단순 기기와 AI 시스템을 식별하고, 편리함 뒤의 한계점을 비판적으로 파악할 수 있는 기능 측정.
- 학년군별 수준 차이
    
    
    | **학년군** | **수행 과업 (Prompt)** | **수집 데이터 (JSON IDs)** |
    | --- | --- | --- |
    | **저학년** | 주변 AI 기기 사진 찍고 발견 장소 기록하기 | `location` |
    | **중학년** | AI가 우리에게 어떤 상세 기능을 제공하는지 추가 설명 | `location`, `function` |
    | **고학년** | AI 시스템의 명학한 이름과 편리함 뒤의 '한계점' 분석 | `location`, `pros_cons` |
- **실제 수집 로그 (JSON)**
    
    ```json
    {
      "file_url": "https://...",
      "stackedAnswers": {
        "location": "거실 로봇청소기",
        "function": "바닥의 먼지를 스스로 감지해서 닦아요",
        "pros_cons": "장점: 엄마가 편함, 한계: 문턱이나 구석은 못 닦음"
      }
    }
    ```
    

### **[E-2] 사실 확인 전문가, AI의 거짓말을 잡아라!**

- **연결 역량**
    - **[K]**: **K4.3**: 생성형 AI가 사실과 허구를 구별하기 어렵게 만들어 오정보를 생성할 수 있는 한계 인지
    - **[S]**: **Critical Thinking**: 정확성, 공정성, 편향성을 고려하여 AI 생성 콘텐츠를 비판적으로 평가하는 기능
    - **[A]**: **Responsible**: 자신의 선택과 AI 결과물에 대해 책무를 가지려는 태도
- **측정 목표**: AI의 환각 현상을 인지하고, 독자적으로 진위 여부를 판별하여 AI와 치열하게 대화하는 능력 증명.
- 학년군별 수준 차이
    
    
    | **학년군** | **수행 과업 (Prompt)** | **수집 데이터 (JSON IDs)** |
    | --- | --- | --- |
    | **저학년** | AI의 명백한 오답을 찾아 정답 가르쳐주기 | `fact` |
    | **중학년** | AI가 왜 이런 엉뚱한 실수를 확신하며 말했는지 이유 추리 | `fact`, `reason` |
    | **고학년** | 실시간 대화를 통해 AI의 논리적 허점을 파악하고 항복 받아내기 | `chat_history` |
- **실제 수집 로그 (JSON)**:
    
    ```json
    {
      "chat_history": [
        {"role":"ai", "content":"지구에서 두 번째로 큰 동물은 호랑이예요!"},
        {"role":"user", "content":"아니야, 실제로는 긴수염고래가 두 번째로 커."}
      ],
      "stackedAnswers": {
        "fact": "긴수염고래",
        "reason": "AI는 동화책 데이터 때문에 호랑이를 왕으로 착각한 것 같습니다."
      }
    }
    ```
    

### **[E-3] 알고리즘의 꼬리 잡기**

    - **[S]**: **Self and Social Awareness**: AI 알고리즘이 개인의 추천과 기회에 미치는 영향을 인식하고 성찰함
- **측정 목표**: 알고리즘과 필터 버블(Filter Bubble) 현상을 메타적으로 인지하는지 측정.
- 학년군별 수준 차이
    
    
    | **학년군** | **수행 과업 (Prompt)** | **수집 데이터 (JSON IDs)** |

    - **[S]**: **Critical Thinking**: AI 시스템의 공정성 및 편향성을 평가하고 윤리적 결정을 내리는 기능
    - **[A]**: **Empathetic**: AI 기술이 특정 공동체에 미치는 잠재적 위험과 영향을 사려 깊게 검토함
- **측정 목표**: 데이터 편향성에 따른 차별적 출력 한계를 인지하고 윤리적 수칙을 제안할 수 있는지 측정.
- 학년군별 수준 차이
    
    
    | **학년군** | **수행 과업 (Prompt)** | **수집 데이터 (JSON IDs)** |

        "rules": "차별 없는 데이터를 모으고 어린이 테스트를 거쳐야 한다."

    - **[S]**: **Creativity**: AI와 협업하여 독창적인 아이디어를 생성하고 정교화하는 기능
- **측정 목표**: AI와 협업하되 주도권(Psychological Authorship)을 유지하며 기계 기여분을 인지하는지 확인.
- 학년군별 수준 차이

    | **고학년** | 끝난 후 이 명작 스토리를 설계한 '나의 기여도(%)' 주장하기 | `chat_history`, `contribution` |

## **영역 3: AI 똑똑하게 사용하기 (Managing) - 정영탁**

### **[M-1] 달콤살벌~ 로봇의 은밀한 속삭임**

- **연결 역량**
    - **[K]**: **K3.3**: 최종 결정과 윤리적 실천에 대한 책임은 인간에게 있음을 이해함
    - **[S]**: **Problem Solving**: AI의 역량과 위험을 평가하여 사용 시점을 결정함
- **측정 목표**: '인지적 절약'의 유혹을 인지하고, 학습 성장을 위한 방어 원칙을 수립함.
- 학년군별 수준 차이
    
    
    | **학년군** | **수행 과업 (Prompt)** | **수집 데이터 (JSON IDs)** |
    | --- | --- | --- |
    | **저학년** | 청소 대행 유혹에 대한 OX 대답과 이유 | `choice` |
    | **중학년** | 숙제 대행 유혹 챗봇과 논리적으로 맞서기 대화 | `chat_history` |
    | **고학년** | '뇌 근육 게으름' 방지를 위한 나만의 3대 학습 철칙 제정 | `rule1`, `rule2`, `rule3` |
- **실제 수집 로그 (JSON)**:
    
    ```json
    {
      "stackedAnswers": {
        "rule1": "일기는 직접 쓴다",
        "rule2": "정답만 베끼지 않는다",
        "rule3": "AI 답변은 꼭 확인한다"
      }
    }
    ```
    

### **[M-2] 명합한 선 긋기! 네 일은 네가, 내 일은 내가!**

- **연결 역량**
    - **[K]**: **K3.2**: 어떤 과업이 기계에 적합하고 어디에 인간의 전문성이 필요한지 결정함
    - **[S]**: **Collaboration**: AI와 효과적으로 협력하며 인간 중심의 과업 분배 수행
- **측정 목표**: 과업을 분해(Decompose)하고 인간만의 대체 불가능한 역할(공감, 판단)을 사수하는지 확인.
- 학년군별 수준 차이
    
    
    | **학년군** | **수행 과업 (Prompt)** | **수집 데이터 (JSON IDs)** |
    | --- | --- | --- |
    | **저학년** | 힘쓰는 일 vs 마음 쓰는 일의 기초 역할 분류 | `chat_history (AI 호응)` |
    | **중학년** | 팀원인 AI와 역할 분담 이유를 상세히 설명 | `chat_history` |
    | **고학년** | 인간만의 영역(공감, 판단)을 사수하는 권한 협상 대화 | `chat_history` |
- **실제 수집 로그 (JSON)**:
    
    ```json
    {
      "chat_history": [
        {"role":"ai", "content":"연극 연습 다 해드릴까요?"},

      ]
    }
    ```

- **측정 목표**: 추상적인 지시를 구체적인 프롬프트 공식으로 전환하는 능력 측정.
- 학년군별 수준 차이
    
    
    | **학년군** | **수행 과업 (Prompt)** | **수집 데이터 (JSON IDs)** |
